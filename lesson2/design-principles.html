Alignment

VASA Fitness

www.vasafitness.com

This site is an example of Alignment. This site has many pictures and is laid out in a way that allows the information to be seperate but also highlighted.

Whitespace

Brighton Ski Resort

www.brightonresort.com

This site is an example of Whitespace. There are smart uses of whitespace on this page and it allows for things to pop out and to flow nicely.

Contrast

Utah DMV

www.dmv.utah.gov

This site is an example of Contrast. This site uses contrast in a way to allow for aspects to pop out without relying on just whitespace. This site uses multiple colors and is still readable.
